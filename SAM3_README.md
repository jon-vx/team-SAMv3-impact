# SAM3 Vanilla Inference

Untrained SAM3 (`facebook/sam3`) via HuggingFace Transformers. Bounding boxes are automatically generated by the vendor UNet (INIA) rather than hardcoded. Same API pattern as MedSAM3.

## Setup

See [README.md](README.md) for full setup. Requires a HuggingFace token:

```bash
echo "HF_TOKEN=hf_xxxxxxxxxxxxx" > .env.local
```

For UNet bbox generation:
```bash
pip install tensorflow tensorflow-metal keras-unet-collection
```

## How it works

1. The vendor UNet (INIA) predicts a rough segmentation mask and extracts a bounding box from it
2. The box is scaled from the UNet's 320×320 space back to the original image dimensions
3. SAM3 uses that box as a prior prompt to produce a fine segmentation mask

## Inference

```python
from impact_team_2.inference._inference_sam3 import build_predictor, load_unet_weights

# Load UNet weights (cached — only loads once)
unet = load_unet_weights("checkpoints/best_unetp.weights.h5")

# Build SAM3 predictor with auto bbox generation
predictor = build_predictor(unet_model=unet)
result = predictor("scan.png", text_prompt="spleen organ in ultrasound")

# Or pass unet_weights directly to build_predictor
predictor = build_predictor(unet_weights="checkpoints/best_unetp.weights.h5")
```

`image` accepts a file path, PIL Image, or numpy array (HxW or HxWx3).

### Output

```python
{
    "boxes":          np.ndarray,   # (N, 4) bounding boxes from SAM3 masks
    "scores":         np.ndarray,   # (N,) confidence scores
    "masks":          np.ndarray,   # (N, H, W) bool masks
    "num_detections": int,
    "image":          PIL.Image,
}
```

Returns `None` for boxes/scores/masks if no detections pass the threshold.

### Parameters

| Parameter | Default | Description |
|---|---|---|
| `unet_weights` | `None` | Path to `.h5` weights file — builds and caches UNet on first call |
| `unet_model` | `None` | Pre-built Keras UNet model (alternative to `unet_weights`) |
| `text_prompt` | required | Text description of the target region |
| `box` | `None` | Manual bbox `[x1, y1, x2, y2]` — overrides UNet bbox if provided |
| `threshold` | `0.5` | Confidence threshold for both SAM3 and UNet |

## Example Script

```bash
# Run with trained UNet (default — loads checkpoints/best_unetp.weights.h5)
venv/bin/python examples/test_sam3_inf.py

# Run on a specific image
venv/bin/python examples/test_sam3_inf.py path/to/image.png

# Use a different weights file
venv/bin/python examples/test_sam3_inf.py --unet path/to/model.h5

# Retrain UNet from scratch and save new weights
venv/bin/python examples/test_sam3_inf.py --train
```

Saves purple overlay PNGs to `runs/`.

## Differences from MedSAM3

| | SAM3 | MedSAM3 |
|---|---|---|
| Model | `facebook/sam3` (HuggingFace) | `lal-Joey/MedSAM3_v1` (native sam3 + LoRA) |
| Fine-tuned | No — vanilla weights | Yes — LoRA fine-tuned on medical images |
| Bounding box | Auto-generated via vendor UNet (INIA) | Not required — detected from text prompt |
| Device | CUDA / MPS / CPU | CUDA / CPU |

## Post-processing

Each mask undergoes morphological closing (7×7, 2 iterations) followed by largest connected component selection.
